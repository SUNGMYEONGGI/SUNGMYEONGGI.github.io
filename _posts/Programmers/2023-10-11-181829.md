# 이차원 배열 대각선 순회하기
## 📃 문제설명

2차원 정수 배열 ```board```와 정수 ```k```가 주어집니다.

```i``` + ```j``` &lt;= ```k```를 만족하는 모든 (```i```, ```j```)에 대한 ```board[i][j]```의 합을 return 하는 solution 함수를 완성해 주세요.

## ❌ 제한사항
<ul>
<li>1 ≤ board의 길이 ≤ 100</li>
<li>1 ≤ board[i]의 길이 ≤ 100

<ul>
<li>1 ≤ board[i][j] ≤ 10,000</li>
<li>모든 board[i]의 길이는 같습니다.</li>
</ul></li>
<li>0 ≤ k &lt; board의 길이 + board[i]의 길이</li>
</ul>


## 🙆🏻‍♂️ 문제 답안
- 내가 제출한 답
    ```python
    def solution(board, k):
    answer = 0
    for i in range(len(board)):
        for j in range(len(board[i])):
            if i+j <= k:
                answer += board[i][j]
    return answer
    ``` 

- [JavaScript 풀이](https://velog.io/@ovzip/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%A8%B8%EC%8A%A4-%EC%BD%94%EB%94%A9-%EA%B8%B0%EC%B4%88-%ED%8A%B8%EB%A0%88%EC%9D%B4%EB%8B%9D-%EC%9D%B4%EC%B0%A8%EC%9B%90-%EB%B0%B0%EC%97%B4-%EB%8C%80%EA%B0%81%EC%84%A0-%EC%88%9C%ED%9A%8C%ED%95%98%EA%B8%B0#-%EB%82%98%EC%9D%98-%ED%92%80%EC%9D%B4)
    ```javascript
    function solution(board, k) {
        var answer = 0;
        for (let i = 0; i < board.length; i += 1){ 
            for (let j = 0; j < board[0].length; j += 1) {
                if (i + j <= k) {
                    answer += board[i][j];
                }
            }
        }
        return answer;
    }
    ```

- [C++ 풀이](https://fate7902.tistory.com/m/525)
    ```cpp
    #include <string>
    #include <vector>
    
    using namespace std;
    
    int solution(vector<vector<int>> board, int k) {
        int answer = 0;
        for(int i = 0; i < board.size(); ++i){
            for(int j = 0; j < board[i].size(); ++j){
                if(i + j <= k) answer += board[i][j];
                else break;
            }
        }
        return answer;
    }
    ```
