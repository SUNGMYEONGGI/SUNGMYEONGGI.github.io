# 집합 자료형
## 1.집합 자료형은 어떻게 만들까?
집합 자료형은 다음과 같이 set 키워드를 사용해 만들 수 있다.
```python
s1 = set([1, 2, 3])
s1 # {1, 2, 3}
```
위와 같이 set()의 괄호 안에 리스트를 입력하여 만들거나 다음과 같이 문자열을 입력하여 만들 수도 있다.

```python
s2 = set("Hello")
s2 # {'e', 'H', 'l', 'o'}
```
비어 있는 집합 자료형은 s = set()로 만들 수 있다.

## 2. 집합 자료형의 특징
- 중복을 허용하지 않는다.
- 순서가 없다.

## 3. 교집합, 합집합, 차집합 구하기
set 자료형을 정말 유용하게 사용하는 경우는 교집합, 합집합, 차집합을 구할 때이다.
```python
s1 = set([1, 2, 3, 4, 5, 6])
s2 = set([4, 5, 6, 7, 8, 9])
```

### 3-1. 교집합 구하기
‘&’를 이용하면 교집합을 간단히 구할 수 있다. `intersection` 함수를 사용해도 결과는 동일하다.
```python
s1 & s2 # {4, 5, 6}
s1.intersection(s2) # {4, 5, 6}
```

### 3-2. 합집합 구하기
‘|’를 사용하면 합집합을 구할 수 있다. `union`을 사용해도 결과는 동일하다.1개씩만 표현된다.
```python
s1 | s2 # {1, 2, 3, 4, 5, 6, 7, 8, 9}
s1.union(s2) # {1, 2, 3, 4, 5, 6, 7, 8, 9}
```

### 3-3. 차집합 구하기
'-'를 사용하면 차집합을 구할 수 있다. `difference`를 사용해도 결과는 동일하다.
```python
s1 - s2 # {1, 2, 3}
s2 - s1 # {8, 9, 7}
s1.difference(s2) # {1, 2, 3}
s2.difference(s1) # {8, 9, 7}
```

## 4. 집합 자료형 관련 함수
### 4-1. 값 1개 추가하기 - `add`
이미 만들어진 set 자료형에 값을 추가할 수 있다. 1개의 값만 추가할 때는 다음과 같이 하면 된다.
```python
s1 = set([1, 2, 3])
s1.add(4)
s1 # {1, 2, 3, 4}
```

### 4-2. 값 여러 개 추가하기 - `update`
여러 개의 값을 한꺼번에 추가할 때는 다음과 같이 하면 된다.
```python
s1 = set([1, 2, 3])
s1.update([4, 5, 6])
s1 # {1, 2, 3, 4, 5, 6}
```

### 4-3. 특정 값 제거하기 - `remove`
특정 값을 제거하고 싶을 때는 다음과 같이 하면 된다.

```python
s1 = set([1, 2, 3])
s1.remove(2)
s1
{1, 3}
```
