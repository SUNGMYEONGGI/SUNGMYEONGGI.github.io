# 딕셔너리 자료형

## 1. 딕셔너리란?
딕셔너리는 단어 그대로 ‘사전’이라는 뜻이며, 즉 "people"이라는 단어에 "사람", "baseball"이라는 단어에 "야구"라는 뜻이 부합되듯이 딕셔너리는 Key와 Value를 한 쌍으로 가지는 자료형이다.

![](https://wikidocs.net/images/page/16/02_5_baseball.png)

## 2. 딕셔너리는 어떻게 만들까?
Key와 Value의 쌍 여러 개가 { }로 둘러싸여 있다. 각각의 요소는 Key : Value 형태로 이루어져 있고 쉼표(,)로 구분되어 있다.
```
{Key1:Value1, Key2:Value2, Key3:Value3, ...}
```
```python
dic = {'name':'pey', 'phone':'0119993323', 'birth': '1118'}
```

딕셔너리 dic의 정보

| Key | Value |
|:---:|:---:|
| name | 'pey' |
| phone | '0119993323' |
| birth | '1118' |

## 3. 딕셔너리 쌍 추가, 삭제하기

### 3-1. 딕셔너리 쌍 추가하기
```python
# Key를 지정하고 Value를 대입
a = {1: 'a'}
a[2] = 'b'
a # {1: 'a', 2: 'b'}

# 딕셔너리 a에 'name': 'pey'라는 쌍을 추가
a['name'] = 'pey'
a # {1: 'a', 2: 'b', 'name': 'pey'}
a[3] = [1,2,3]
a # {1: 'a', 2: 'b', 'name': 'pey', 3: [1, 2, 3]}
```

### 3-2. 딕셔너리 요소 삭제하기
del a[key]를 입력하면 지정한 Key에 해당하는 {Key: Value} 쌍이 삭제된다.
```python
del a[1]
a # {2: 'b', 'name': 'pey', 3: [1, 2, 3]}
```

## 4. 딕셔너리를 사용하는 방법

### 4-1. 딕셔너리에서 Key를 사용해 Value 얻기
Key의 Value를 얻기 위해 grade['pey']를 사용한 것처럼 어떤 Key의 Value를 얻기 위해서는 '딕셔너리_변수_이름[Key]'를 사용해야 한다.
```python
grade = {'pey': 10, 'julliet': 99}
grade['pey'] # 10
grade['julliet'] # 99
```

```python
dic = {'name':'pey', 'phone':'010-9999-1234', 'birth': '1118'}
dic['name'] # 'pey'
dic['phone'] # '010-9999-1234'
dic['birth'] # '1118'
```

### 4-2. 딕셔너리 만들 때 주의할 사항
딕셔너리에서 Key는 고유한 값이므로 중복되는 Key 값을 설정해 놓으면 하나를 제외한 나머지 것들이 모두 무시된다는 점에 주의해야 한다. 다음 예에서 볼 수 있듯이 동일한 Key가 2개 존재할 경우, 1: 'a' 쌍이 무시된다.

```python
a = {1:'a', 1:'b'}
a # {1: 'b'}
```
- 딕셔너리에는 동일한 Key가 중복으로 존재할 수 없다.
- Key에 리스트는 쓸 수 없다는 것이다. 하지만 튜플은 Key로 쓸 수 있다. 

## 5. 딕셔너리 관련 함수
딕셔너리를 자유자재로 사용하기 위해 딕셔너리가 자체적으로 가지고 있는 관련 함수를 사용해 보자.
```python
a = {'name': 'pey', 'phone': '010-9999-1234', 'birth': '1118'}
```

### 5-1. Key 리스트 만들기 - `keys`
`a.keys()`는 딕셔너리 a의 Key만을 모아 dict_keys 객체를 리턴한다.
```python
a.keys() # dict_keys(['name', 'phone', 'birth'])
```

### 5-2. Value 리스트 만들기 - `values`
`values` 함수를 호출하면 dict_values 객체를 리턴한다.
```python
a.values() # dict_values(['pey', '010-9999-1234', '1118'])
```

### 5-3. Key, Value 쌍 얻기 - `items`
`items` 함수는 Key와 Value의 쌍을 튜플로 묶은 값을 dict_items 객체로 리턴한다.
```python
a.items() # dict_items([('name', 'pey'), ('phone', '010-9999-1234'), ('birth', '1118')])
```


### 5-4. Key: Value 쌍 모두 지우기 - `clear`
`clear` 함수는 딕셔너리 안의 모든 요소를 삭제한다.
```python
a.clear()
a # {}
```

### 5-5. Key로 Value 얻기 - `get`
`get(x)` 함수는 x라는 Key에 대응되는 Value를 리턴한다.
```python
a = {'name': 'pey', 'phone': '010-9999-1234', 'birth': '1118'}
a.get('name') # 'pey'
a.get('phone') # '010-9999-1234'
```

### 5-6. 해당 Key가 딕셔너리 안에 있는지 조사하기 - `in`
'name' in a를 호출하면 참(True)을 리턴한다. 이와 반대로 'email'은 a 딕셔너리 안에 존재하지 않는 Key이므로 거짓(False)을 리턴한다.
```python
a = {'name':'pey', 'phone':'010-9999-1234', 'birth': '1118'}
'name' in a # True
'email' in a # False
```
