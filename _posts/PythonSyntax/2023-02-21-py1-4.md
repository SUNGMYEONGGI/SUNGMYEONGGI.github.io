# 튜플 자료형

## 1. 튜플은 어떻게 만들까?
튜플은 다음과 같이 여러 가지 모습으로 생성할 수 있다.
```python
t1 = ()
t2 = (1,)
t3 = (1, 2, 3)
t4 = 1, 2, 3
t5 = ('a', 'b', ('ab', 'cd'))
```

## 2. 튜플의 요솟값을 지우거나 변경하려고 하면 어떻게 될까?
`tuple`과 `list`의 가장 큰 차이는 요솟값을 변화시킬 수 있는지의 여부이다. 즉, 리스트의 요솟값은 변화가 가능하고 튜플의 요솟값은 변화가 불가능하다.

### 2-1. 튜플 요솟값을 삭제하려 할 때
튜플의 요소를 리스트처럼 del 함수로 지우려고 한 예이다. 튜플은 요솟값을 지울 수 없다는 오류 메시지를 확인할 수 있다.
```python
>>> t1 = (1, 2, 'a', 'b')
>>> del t1[0]
Traceback (most recent call last):
File "<stdin>", line 1, in <module>
TypeError: 'tuple' object doesn't support item deletion
```

### 2-2. 튜플 요솟값을 변경하려 할 때
튜플의 요솟값을 변경하려고 해도 오류가 발생하는 것을 확인할 수 있다.
```python
t1 = (1, 2, 'a', 'b')
t1[0] = 'c'
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: 'tuple' object does not support item assignment
```

## 3. 튜플 다루기
튜플은 요솟값을 변화시킬 수 없다는 점만 제외하면 리스트와 완전히 동일하므로 간단하게만 살펴보자. 

### 3-1. 인덱싱하기
문자열, 리스트와 마찬가지로 t1[0], t1[3]처럼 인덱싱이 가능하다.
```python
t1 = (1, 2, 'a', 'b')
t1[0] # 1
t1[3] # 'b'
```

### 3-2. 슬라이싱하기
t1[1]부터 튜플의 마지막 요소까지 슬라이싱하는 예이다.
```python
t1 = (1, 2, 'a', 'b')
t1[1:] # (2, 'a', 'b')
```

### 3-3. 튜플 더하기&곱하기
튜플을 더하기&곱하기 방법을 보여 주는 예이다.
```python
t1 = (1, 2, 'a', 'b')
t2 = (3, 4)
t3 = t1 + t2
t3 # (1, 2, 'a', 'b', 3, 4)

t4 = (3, 4)
t5 = t2 * 3
t5 # (3, 4, 3, 4, 3, 4)
```
