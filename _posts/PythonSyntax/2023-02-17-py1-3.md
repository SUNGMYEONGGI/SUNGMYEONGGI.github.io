# 리스트 자료형

## 1. 리스트는 어떻게 만들고 사용할까?
리스트를 사용하면 1, 3, 5, 7, 9의 숫자 모음을 다음과 같이 간단하게 표현할 수 있다. 리스트를 만들 때는 대괄호([])로 감싸 주고 각 요솟값은 쉼표(,)로 구분해 준다.
```python
odd = [1, 3, 5, 7, 9]
```

여러 가지 리스트의 생김새는 다음과 같다.
```python
a = []
b = [1, 2, 3]
c = ['Life', 'is', 'too', 'short']
d = [1, 2, 'Life', 'is']
e = [1, 2, ['Life', 'is']] # 리스트는 어떠한 자료형도 포함할 수 있음
```

## 2. 리스트의 인덱싱과 슬라이싱
리스트도 문자열처럼 인덱싱과 슬라이싱이 가능하다.

### 2-1. 리스트의 인덱싱
리스트 역시 문자열처럼 인덱싱을 적용할 수 있다. 먼저 a 변수에 [1, 2, 3] 값을 설정한다.
```python
a = [1, 2, 3]

# 리스트 a의 첫 번째 요소인 a[0]이라는 것은 1이라는 값을 가지고 있다.
a[0] # 1

# 리스트의 첫 번째 요소인 a[0]과 세 번째 요소인 a[2]의 값을 더한 것이다.
a[0] + a[2] # 4

# 문자열에서와 마찬가지로 리스트 a의 마지막 요솟값이다.
a[-1] # 3
```

### 2-2. 리스트 내부에 리스트가 존재할 경우
이번에는 다음 예처럼 리스트 a를 숫자 1, 2, 3과 또 다른 리스트인 ['a', 'b', 'c']를 포함하도록 만들어 보자.
```python
a = [1, 2, 3, ['a', 'b', 'c']]
```

```python
a[0] # 1
a[-1] # ['a', 'b', 'c']
a[3] # ['a', 'b', 'c']
```

a[-1]은 마지막 요솟값 ['a', 'b', 'c']를 나타내며 a[3]은 리스트 a의 네 번째 요소를 나타내기 때문에 마지막 요소를 나타내는 a[-1]과 동일한 결괏값을 보여준다.

#### 🙋🏻‍♂️ 리스트 a에 포함된 ['a', 'b', 'c'] 리스트에서 'a' 값을 인덱싱을 사용해 끄집어 낼 수 있는 방법은 없을까?
```python
# a[-1]의 리스트에서 첫 번째 요소를 불러오기 위해 [0]을 붙여 준 것
a[-1][0] # 'a'
```

### 2-3. 리스트의 슬라이싱
문자열과 마찬가지로 리스트에서도 슬라이싱 기법을 적용할 수 있다. 슬라이싱은 ‘잘라낸다’라는 뜻이라고 했다. 리스트의 슬라이싱에 대해서 살펴보자.
```python
a = [1, 2, 3, 4, 5]

a[0:2] # [1, 2]
a[:2] # [1, 2]
a[2:] # [3, 4, 5]
```

## 3. 리스트 연산하기

### 3-1. 리스트 더하기(+)
```python
a = [1, 2, 3]
b = [4, 5, 6]
a + b # [1, 2, 3, 4, 5, 6]
```

### 3-2. 리스트 반복하기(*)
```python
a = [1, 2, 3]
a * 3 # [1, 2, 3, 1, 2, 3, 1, 2, 3]
```

#### 🙅🏻‍♂️ 초보자가 범하기 쉬운 리스트 연산 오류
다음 소스 코드를 입력했을 때 결괏값은 어떻게 나올까?
```python
a = [1, 2, 3]
a[2] + "hi"
```
```bash
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: unsupported operand type(s) for +: 'int' and 'str'
```

🚫 오류의 원인은 무엇일까? 🚫
```python
# '3hi'를 만들고 싶다면 숫자 3을 문자 '3'으로 바꾸어야 함
str(a[2]) + "hi" # 3hi
```

## 4. 리스트의 수정과 삭제

### 4-1. 리스트의 값 수정하기
```python
a = [1, 2, 3]
a[2] = 4
a # [1, 2, 4]
```

### 4-2. del 함수를 사용해 리스트 요소 삭제하기
del a[x]는 x번째 요솟값을 삭제한다. 
```python
a = [1, 2, 3]
del a[1]
a # [1, 3]
```

## 5. 리스트 관련 함수

### 5-1. 리스트에 요소 추가하기 - `append`
append의 사전적 의미는 ‘덧붙이다, 첨부하다’이다. append(x)는 리스트의 맨 마지막에 x를 추가하는 함수이다.
```python
a = [1, 2, 3]
a.append(4)
a # [1, 2, 3, 4]
```
리스트 안에는 어떤 자료형도 추가할 수 있다. 다음 예는 리스트에 다시 리스트를 추가한 결과이다.

```python
a.append([5, 6])
a # [1, 2, 3, 4, [5, 6]]
```

### 5-2. 리스트 정렬 - `sort`
```python
# 리스트의 요소를 순서대로 정렬해 준다.
a = [1, 4, 3, 2]
a.sort()
a # [1, 2, 3, 4]

# 문자 역시 알파벳 순서로 정렬할 수 있다.
a = ['a', 'c', 'b']
a.sort()
a # ['a', 'b', 'c']
```

### 5-3. 리스트 뒤집기 - `reverse`
reverse 함수는 리스트를 역순으로 뒤집어 준다.
```python
a = ['a', 'c', 'b']
a.reverse()
a # ['b', 'c', 'a']
```

### 5-4. 인덱스 반환 - `index`
index(x) 함수는 리스트에 x 값이 있으면 x의 인덱스 값(위칫값)을 리턴한다. 리스트 a에 있는 숫자 3의 위치는 a[2]이므로 2, 숫자 1의 위치는 a[0]이므로 0을 리턴한다.
```python
a = [1, 2, 3]
a.index(3) # 2
a.index(1) # 0
```

### 5-5. 리스트에 요소 삽입 - `insert`
insert(a, b)는 리스트의 a번째 위치에 b를 삽입하는 함수이다.
```python
a = [1, 2, 3]
# 첫 번째 요소인 a[0] 위치에 값 4를 삽입
a.insert(0, 4)
a # [4, 1, 2, 3]

# 네 번째 요소 위치에 값 5를 삽입
a.insert(3, 5)
a # [4, 1, 2, 5, 3]
```

### 5-6. 리스트 요소 제거 - `remove`
remove(x)는 리스트에서 첫 번째로 나오는 x를 삭제하는 함수이다. a가 3이라는 값을 2개 가지고 있을 경우, 첫 번째 3만 제거되는 것을 알 수 있으며 remove(3)을 한 번 더 실행하면 나머지 3이 삭제된다.
```python
a = [1, 2, 3, 1, 2, 3]
a.remove(3)
a # [1, 2, 1, 2, 3]

a.remove(3)
a # [1, 2, 1, 2]
```

### 5-7. 리스트 요소 끄집어 내기 - `pop`
pop()은 리스트의 맨 마지막 요소를 리턴하고 그 요소는 삭제한다.
```python
a = [1, 2, 3]
a.pop() # 3
a # [1, 2]
```

### 5-8. 리스트에 포함된 요소 x의 개수 세기 - `count`
count(x)는 리스트 안에 x가 몇 개 있는지 조사하여 그 개수를 리턴하는 함수이다.
```python
a = [1, 2, 3, 1]
a.count(1) # 2
```

### 5-9. 리스트 확장 - `extend`
extend(x)에서 x에는 리스트만 올 수 있으며 원래의 a 리스트에 x 리스트를 더하게 된다.
```python
a = [1, 2, 3]
a.extend([4, 5])
a # [1, 2, 3, 4, 5]

b = [6, 7]
a.extend(b)
a # [1, 2, 3, 4, 5, 6, 7]
```
