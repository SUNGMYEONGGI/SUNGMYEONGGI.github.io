## RunPod 사용법
![](https://velog.velcdn.com/images/sungmyeonggi/post/e2330d0a-7f71-4991-9552-905dcf12e982/image.png)

GPU 대여 클라우드 서비스 - [Runpod](https://www.runpod.io/)

![](https://velog.velcdn.com/images/sungmyeonggi/post/4d87af5f-23cf-4547-9ecc-e03d1e4a0067/image.png)

1) 먼저, 계정을 만들고 돈을 충전합니다.
2) 코랩하고 유사하며 다른점은 코랩은 VRAM (유료/무료)12GB/20GB 제한인데 RunPod는 그래픽카드 VRAM을 그대로 쓴다는 것
3) 사용하는 모델에 맞게 그래픽카드를 Deploy 해주세요.
4) 보안 딱히 상관없으면 Community 사용, 신경쓰이면 Secure 사용

![](https://velog.velcdn.com/images/sungmyeonggi/post/93894291-395c-4c25-b020-49011162b98d/image.webp)

6) Select a Template -> RunPod Pytorch 2 선택
7) Container Disk & Volume Disk 상관없음

![](https://velog.velcdn.com/images/sungmyeonggi/post/1149e2e8-44a8-4da2-86fc-b16e909f3d3b/image.png)

8) 계속 진행 Deploy하면 이런 화면이 뜸
9) Connect Button Click

![](https://velog.velcdn.com/images/sungmyeonggi/post/ea007c4c-1851-4fc6-b1c6-4ecc86598c26/image.png)

10) Jupyter Lab으로 접속합니다.

![](https://velog.velcdn.com/images/sungmyeonggi/post/253bc676-3b63-4ebf-9eb6-3e4f5a05c94f/image.png)

11) 노트북을 하나 만들어줍니다.
12 아래 설치파일을 하나씩 설치해줍니다.

```
# 우바부가 설치
!git clone https://github.com/oobabooga/text-generation-webui
%cd text-generation-webui
!pip install -r requirements.txt
```
```
# peft 버전 다운그레이드 - 0.3.0은 학습시 메모리터짐 0.2.0으로 다운그레이드 함
!pip uninstall -y peft
!pip install peft==0.2.0
```
```
# oobabooga 실행, ChatMode 실행
!python server.py --chat --no-stream --share --listen
```

![](https://velog.velcdn.com/images/sungmyeonggi/post/5d13cdb9-41f9-40bf-bfad-a811d3ed851f/image.png)

13) 설치 후 xxxxx.gradio.live라고 뜨면 클릭

![](https://velog.velcdn.com/images/sungmyeonggi/post/f250399c-0bac-48a9-b1d8-ce1dd2f479dd/image.webp)

14) 탭에서 Model을 클릭

![](https://velog.velcdn.com/images/sungmyeonggi/post/b5fd83ae-6293-4d59-a6bb-9d561f624e5e/image.png)

15) auto-devices & load-in-8bit 선택
16) Hugging Face에서 학습 시킬 모델을 Download custom model or LoRA 부분에 링크 삽입
17) Download 후 노트북에 다시 들어가면 설치되는것이 보임
18) Model 새로고침 후 설치받은 Model 선택 후 학습하러 ㄱㄱ

![](https://velog.velcdn.com/images/sungmyeonggi/post/f34943e4-5be5-4316-aa01-41ef651f8686/image.png)

19) workspace/text-generation-webui/training/datasets에 님들 데이터셋 업로드 

![](https://velog.velcdn.com/images/sungmyeonggi/post/363e3d43-c08a-4524-9550-df3fbfb9c4b6/image.png)![](https://velog.velcdn.com/images/sungmyeonggi/post/d87a90ab-a526-4e92-8cfc-9aba40f163cd/image.png)

20) 탭에서 Traning에 들어가면 WebUI가 뜸
21) 파일 이름을 정해주고 필요시 Hyperparameter를 조정함
22) Formatted Datset & Raw text file에서 데이터셋 업로드
23) Start LoRA Training하면 돈이 녹으면서 나만의 대규모언어모델이 탄생하는 것
