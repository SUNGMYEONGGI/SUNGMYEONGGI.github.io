# 회귀(Regression)이란?
회귀 모델은 주어진 데이터를 학습시켜 연속적인 예측값을 출력하는 모델이다.

회귀모델는 주어진 데이터를 가장 잘 설명하는 하나의 직선을 찾는 것이다. 이 직선을 찾기 위해 주어진 데이터와 직선 사이의 오차를 최소화하는 직선을 찾는 것이 목표이다.
회귀 모델은 크게 선형 회귀모델(Linear Regression)과 비선형 회귀모델(Non-Linear Regression)로 나뉜다. 여기서 선형과 비선형은 아래의 수식을 만족시키냐 만족시키지 못 하느냐에 따라 나뉩니다.

$$f(u+v) = f(u) + f(v)$$

$$f(c*u) = c*f(u)$$

$$(u, v : vaules, c : constant)$$

대표적인 선형 회귀모델은 아래와 같은 모양이다.

$$y = w_1x_1 + w_2x_2 + ... + w_nx_n + b$$

선형회귀는 변수의 개수만 다를 뿐 모두 동일한 모양을 가지고 선형이라는 성질을 가지고 있는 것에 변함이 없다. 모델의 형태를 보면 결국 직선, 평면, 4차원, 초평면이 될 것이다. 따라서 선형 회귀모델은 데이터를 표현하는 차원의 공간에서, 데이터들과 이루는 오차가 가장 작은 직선 혹은 평면을 찾는 모델이다.

여기서 데이터가 2차원의 공간에서 표현이 되어 회귀 모델이 단순한 직선 형태가 되면 단순 선형회귀(Simple Linear Regression), 이상의 공간에서 평면 혹은 초평면 형태가 된다면 다중 선형회귀(Multiple Linear Regression)라고 부른다.

## 단순 선형회귀(Simple Linear Regression)
먼저, 단순 선형회귀의 코드와 시각화 이미지를 살펴보자.
```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import linregress

# 데이터 생성
np.random.seed(0)
x = np.random.rand(100) * 10
y = 2.5 * x + np.random.randn(100)  # 노이즈

# 선형 회귀 모델 적합
slope, intercept, r_value, p_value, std_err = linregress(x, y)

# 회귀 직선 계산
regression_line = slope * x + intercept

# 데이터 시각화
plt.scatter(x, y, s=20, label='Data Points')
plt.plot(x, regression_line, color='red', label='Regression Line')
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Simple Linear Regression')
plt.legend()
plt.show()
```

단순 선형회귀 시각화 이미지에 대해 간단히 설명해보자.
- 데이터 : 이미지 위 파란색 점들이 각각의 데이터를 나타낸다. 이 데이터들은 2차원의 공간에 표현된다.
- 회귀선 : 이미지 위 빨간색 직선이 회귀선이다. 이 선은 모든 데이터의 오차를 최소한 부분에 그려진다.

## 비선형 회귀모델(Non-Linear Regression)
비선형 회귀모델은 X, Y값의 선형관계가 아닌 비선형 관계일 때 사용하는 분석 방법이다. 바로 코드를 이용해 시각화를 해보자.
```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import linregress

# 데이터 생성
np.random.seed(0)
x = np.random.rand(100) * 10 
y_linear = 2.5 * x + np.random.randn(100)  # 노이즈
y_nonlinear = 0.5 * x**2 + 2.5 * x + np.random.randn(100)  # 노이즈

# 선형 회귀 모델 적합
slope, intercept, r_value, p_value, std_err = linregress(x, y_linear)
regression_line = slope * x + intercept

# 데이터 시각화
plt.figure(figsize=(14, 6))

# 선형 데이터 시각화
plt.subplot(1, 2, 1)
plt.scatter(x, y_linear, s=20, label='Linear Points')
plt.plot(x, regression_line, color='red', label='Linear Line')
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Simple Linear Regression')
plt.legend()

# 비선형 데이터 시각화
plt.subplot(1, 2, 2)
plt.scatter(x, y_nonlinear, s=20, label='Non-Linear Points')
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Nonlinear Data')
plt.legend()

plt.show()
```
