# 퍼셉트론(Perceptron)이란?
퍼셉트론이란 다수의 신호를 입력으로 받아 하나의 신호를 출력하는 알고리즘이다. 이때, 신호는 '흐른다(1)' 혹은 '안 흐른다(0)'의 두 가지 값을 가질 수 있다. 이러한 퍼셉트론은 다수의 입력 신호에 각각의 가중치를 부여하고, 이들을 모두 더한 값이 정해진 한계를 넘어설 때만 1을 출력한다. 이 한계를 임계값이라고 하며, 이 임계값을 넘을 때만 신호를 출력한다.

퍼셉트론은 크게 단층 퍼셉트론과 다층 퍼셉트론으로 나뉜다.

## 단층 퍼셉트론(Single-Layer Perceptron)
단층 퍼셉트론은 입력층과 출력층으로만 이루어진 퍼셉트론을 말한다. 논리 회로로 표현하면 AND, OR, NAND 게이트를 표현 할수 있다.

<img src="https://wikidocs.net/images/page/24958/andgraphgate.PNG" width="350px" alt="AND 게이트">

먼저, AND 게이트는 두 입력이 모두 1일 때만 1을 출력하고, 그 외에는 0을 출력하게 되며 직선 하나로 나눠지는 영역을 만들 수 있다. 또 다른 경우로 OR, NAND 게이트도 있다.

<img src="https://wikidocs.net/images/page/24958/oragateandnandgate.PNG" width="400px" alt="OR, NAND 게이트">

## 다층 퍼셉트론(Multi-Layer Perceptron)
인공지능의 1차 암흑기는 다층 퍼셉트론의 베타적논리합(XOR) 때문이라고 해도 과언이 아니다. XOR 게이트는 단층 퍼셉트론과 다르게 직선 하나로 나눠지는 영역을 만들 수 없기 때문에 단층 퍼셉트론으로는 XOR 게이트를 표현할 수 없다. 사진으로 살펴보자.

<img src="https://wikidocs.net/images/page/24958/xorgraphandxorgate.PNG" width="400px" alt="XOR 게이트">

AND, OR, NAND 게이트를 결합하여 XOR 게이트를 구현하기 위해 등장한 것이 은닉층(hidden layer)이다. 은닉층을 추가하면 XOR 게이트를 표현할 수 있다.

<img src="https://wikidocs.net/images/page/24958/perceptron_4image.jpg" width="300px" alt="다층 퍼셉트론">

### XOR Gate 구현식
```python
import numpy as np

def AND(x1, x2):
    x = np.array([x1, x2])
    w = np.array([0.4, 0.4])
    b = -0.7
    tmp = np.sum(w*x) + b
    if tmp > 0:
        return 1
    else:
        return 0

def NAND(x1, x2):
    x = np.array([x1, x2])
    w = np.array([-0.4, -0.4])
    b = 0.7
    tmp = np.sum(w*x) + b
    if tmp > 0:
        return 1
    else:
        return 0

def OR(x1, x2):
    x = np.array([x1, x2])
    w = np.array([0.4, 0.4])
    b = -0.2
    tmp = np.sum(w*x) + b
    if tmp > 0:
        return 1
    else:
        return 0

def XOR(x1, x2):
    s1 = NAND(x1, x2)
    s2 = OR(x1, x2)
    y = AND(s1, x2)
    return y

XOR(0, 0)  # 0 출력
XOR(1, 0)  # 1 출력
XOR(0, 1)  # 1 출력
XOR(1, 1)  # 0 출력
```
